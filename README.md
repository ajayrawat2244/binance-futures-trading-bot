# Binance Futures Testnet Trading Bot

A simple Python-based trading bot for Binance Futures Testnet.

## Features

- Place MARKET orders
- Place LIMIT orders
- BUY and SELL support
- Input validation
- Logging support
- Error handling
- Structured project architecture
